### 퍼블리싱용 브랜치

##### 화면 및 팝업 확인 경로
- 설정 팝업 : 좌측 상단 톱니바퀴 버튼
- 비밀번호 확인 팝업 : 설정 버튼 클릭
- 북마크 목록 관리 팝업 : 비밀번호 확인 팝업에서 확인 버튼 클릭
  - 북마크 색상 변경 : 북마크 목록 관리 팝업에서 색상 선택
- 로딩 중 팝업 : 메인화면 상단 가운데 로고 클릭
- 글 작성 화면 : 우측 상단 첫번째 버튼(연필 그러진 아이콘) 클릭
  - (모바일) : 햄버거 버튼 > 새노트 클릭
- 글 상세 화면 : 글 목록 중 첫번째 글 클릭
  - (모바일) : 글 목록 중 첫번째 글 클릭
- (모바일) 메뉴 팝업 : 우측 상단 햄버거 버튼 클릭
- (모바일) 북마크 목록 팝업 : 햄버거 버튼 > 북마크 클릭
